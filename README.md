Hotel Service Simulation System

A hotel operations simulation developed in C++ that models customer arrivals, service queues, VIP prioritization, and department management in a hotel environment.

Technologies Used

C++
Object-Oriented Programming (OOP)
Linked Lists
Queue Data Structure
Discrete Event Simulation
HTML, CSS, JavaScript
Statistics Visualization

Features

VIP and Normal customer queues
Front Desk, Concierge, and Restaurant departments
Multi-server processing
Customer abandonment logic
Satisfaction rating system
Real-time statistics dashboard
Interactive GUI with charts and controls

Concepts Applied

Data Structures & Algorithms
Queue Management
Simulation Systems
Software Design
Statistical Analysis

Future Improvements

Dynamic server allocation
Database integration
More realistic arrival distributions
Backend-GUI integration

Team Project

Developed as part of the Data Structures & Algorithms course at E-JUST.
